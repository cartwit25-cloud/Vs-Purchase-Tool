📖 Google Apps Script 設定教學

步驟 1：開啟你的 Google Sheets，點選上方選單「擴充功能」→「Apps Script」

步驟 2：刪除預設程式碼，貼上 google-apps-script.js 的內容

步驟 3：點「部署」→「新增部署作業」→ 類型選「網路應用程式」

步驟 4：「誰可以存取」選「所有人」，點「部署」，複製 Web App URL

步驟 5：將 URL 貼到上方「Apps Script Web App URL」欄位並儲存
